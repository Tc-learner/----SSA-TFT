## 一、项目概述

### 1.1 研究背景

田纳西-伊斯曼（Tennessee Eastman, TE）过程是化工过程控制领域的标准基准测试平台。实际化工生产过程中，工况模态（操作模式）的切换会导致过程变量统计特性发生显著变化。及时、准确地检测模态变化，对过程安全监控和故障预警具有重要意义。

### 1.2 技术路线

本项目提出**SSA + TFT 的在线模态变化检测方法**，核心思路如下：

```
TE多变量时序数据 → SSA奇异谱分析（降噪+特征提取）
    → 滑动窗口构造序列 → TFT预测模型（分位数输出）
    → 逐点预测 + RMSE监控 → 连续超阈值判定模态变化
    → 增量微调 + 重新训练 → 继续在线检测
```

### 1.3 创新点

- **SSA预处理**：利用奇异谱分析对TE过程数据进行降噪和主成分提取，相比直接使用原始变量，提高了预测模型对模态变化的敏感度
- **在线增量学习**：检测器在正常运行期间持续微调模型，模态变化时自动重新训练，适应过程动态特性
- **多模型支持**：实现了5种时序预测模型（LSTM / Attention-LSTM / Robust-LSTM / Control-Gate-LSTM / TFT），可灵活对比选择
- **多策略自适应阈值**：支持静态、动态统计、CUSUM累积和、滚动百分位四种阈值方法

### 3.4 在线检测器（核心算法）

**`OnlineDetector` 类实现了完整的在线模态变化检测流程：**

1. **初始训练**：在数据前 `INIT_TRAIN_SIZE` 个样本上训练初始模型
2. **阈值校准**：在后续 `CALIBRATION_SIZE` 个样本上评估模型，计算动态阈值
3. **逐点检测**：逐样本预测并计算RMSE
4. **模态变化确认**：连续 `CONSECUTIVE_COUNT` 个点超阈值 → 判定模态变化
5. **安全检查点**：模态变化时回滚到过渡前的干净模型并保存
6. **重新训练**：从变化点位置开始重新训练，继续检测
7. **增量微调**：正常预测积累到 batch_size 后微调模型

### 3.5 自适应阈值策略

| 方法 | 配置值 | 原理 | 适用场景 |
|------|--------|------|----------|
| Static | `"static"` | 固定阈值 | 已知正常RMSE范围 |
| Dynamic | `"dynamic"` | `mean + multiplier × std` | 一般情况（默认） |
| CUSUM | `"cusum"` | 累积和检测小幅持续漂移 | 渐变模态变化 |
| Percentile | `"percentile"` | 滚动窗口P99动态阈值 | 噪声水平变化大的场景 |

### 3.6 短过渡检测增强

- [x] **多尺度SSA**：同时运行 W=6/12/24 的SSA，短过渡在细粒度窗口（W=6）中表现为尖峰
- [x] **SSA残差特征**：细-粗粒度重建差异作为辅助检测通道
- [x] **双向残差确认**：模态边界从前后两个方向交叉验证——真正的模态变化两侧都应异常
- [x] **变化点预筛选**：基于ruptures库的PELT算法做粗筛，LSTM确认
- [x] **马氏距离回退**：极短片段（<3点）用统计距离做分类，不静默跳过

### 3.7 模型安全检查点

- [x] 正常段定期保存模型权重到内存
- [x] 模态变化确认时回滚到过渡**前**的干净检查点
- [x] Kendall tau污染趋势检测：RMSE单调递增时跳过微调，避免模型学习过渡数据
- [x] 模态变化后 `clear_session()` 释放TF计算图

### 3.8 评估体系

- [x] **检测精度**：Precision / Recall / F1（带容差窗口的地面真值匹配）
- [x] **检测延迟**：真实过渡到首次检测的样本数（均值/中位数/最大值）
- [x] **阈值敏感性AUC**：扫描THRESHOLD_MULTIPLIER，画TPR vs FPR曲线
- [x] **污染分数**：增量微调模型 vs 从头重训模型的退化比
- [x] **逐模态RMSE稳定性**：每个模态段内的RMSE均值/标准差
